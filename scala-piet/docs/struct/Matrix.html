def updateRow(y: Int, newElems: Seq[A]): Unit

Updates a row of elements at once

Updates a row of elements at once

Value parameters:
newElems

collection of new row elements

y

index of the row to be updated

Note:

If newElems length is smaller than this matrix width, only a part of the row will be updated. If newElems length is larger than this matrix width, whole row will be updated, and some newElems elements will be unused

Example:
val m: Matrix[Int] = Matrix(3, 3)
m.updateRow(0, List(1, 2, 3))     // will put 1 2 3 into row 0
m.updateRow(1, List(1, 2))        // will put 1 2 into first two columns of row 1
m.updateRow(2, List(1, 2, 3, 4))  // will put 1 2 3 into row 2 (number 4 will be unused)
